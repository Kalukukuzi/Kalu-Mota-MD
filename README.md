## 🦇For Termux/Ssh/Ubuntu
```bash
$ pkg upgrade -y && pkg update -y
$ pkg install git -y
$ pkg install nodejs -y
$ pkg install ffmpeg -y
$ pkg install imagemagick -y
$ git clone https://github.com/Kalukukuzi/KALU-MOTA-MD
$ cd KALU-MOTA
$ npm i 
$ npm start
```
## 💟For VPS
